# 📝 Task Matrix Bot

Телеграм-бот для управления задачами по матрице Эйзенхауэра (4 квадранта) + быстрые задачи и напоминания.
Минималистичный интерфейс, удобная навигация, напоминания и ежедневная сводка.

---

## ✨ Возможности
- 4 квадранта: **Срочно/Важно**, **Срочно/Неважно**, **Несрочно/Важно**, **Несрочно/Неважно**
- **Быстрые задачи** (без квадранта)
- Напоминания:
  - за день / за час / за 15 минут
  - свои минуты
  - стандартное напоминание в момент дедлайна (вкл/выкл)
- Сводка **“Сегодня”**
- Поиск по задачам (без учета регистра)
- Архив
- Массовое удаление (быстрые задачи и архив)
- Пост‑дедлайн: сообщение “Задача выполнена?” с быстрыми действиями

---

## 🧠 Логика задач
**Квадрант определяется двумя флагами:**
- Важно (important)
- Срочно (urgent)

**Карта квадрантов:**
- 🟥 Срочно/Важно  
- 🟦 Срочно/Неважно  
- 🟩 Несрочно/Важно  
- 🟨 Несрочно/Неважно  

**Быстрые задачи** создаются отдельно и не попадают в квадранты.

## 🗂 Структура проекта
```
app/
  main.py              # точка входа
  config.py            # настройки
  db.py                # работа с SQLite
  states.py            # FSM состояния
  keyboards.py         # все клавиатуры
  handlers/            # логика бота
  services/            # парсинг времени, напоминания, форматирование
requirements.txt
.env
```

---

## ⚙️ Установка
1. Клонируй проект  
2. Установи зависимости:
```
pip install -r requirements.txt
```

3. Создай `.env` в корне проекта:
```
BOT_TOKEN=твой_токен
```

4. Запуск:
```
python app/main.py
```

---

## 🕒 Таймзона
Пользователь задаёт смещение от UTC:
```
+3   # Москва
+5   # Екатеринбург
-5   # США
```

Без таймзоны бот не позволяет создавать задачи.

---

## 🔔 Напоминания
Формат уведомлений:
```
⚠️ Через 15 минут: <текст задачи>
🕒 16.02 15:00
```

```
🔔 Сейчас: <текст задачи>
🕒 16.02 15:00
```

После дедлайна:
```
⚠️ Дедлайн прошел: <текст>
🕒 16.02 22:47

Задача выполнена?
```

## 🧾 Примечания
- База SQLite создается автоматически при первом запуске (`app.db`).
- Напоминания обрабатываются через polling‑цикл (по умолчанию раз в 30 сек).
- `__pycache__` можно игнорировать — это кэш Python.
